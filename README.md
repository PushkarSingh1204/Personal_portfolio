# PUSHKAR SINGH // PORTFOLIO

> AI & ML ENGINEER. IOT SYSTEMS BUILDER.

Welcome to the source code of my personal portfolio. Built with an industrial, terminal-inspired aesthetic and smooth, high-performance interactions to showcase my work in AI, Machine Learning, and IoT Systems.

## 🚀 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **UI Library:** [React](https://reactjs.org/)
- **Styling:** Custom CSS with CSS Variables for consistent theming
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (Parallax, Scroll-based reveal, Hover effects, Custom Cursor)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Icons:** [Devicon](https://devicon.dev/) & Lucide React

## ✨ Key Features

- **Terminal Aesthetic:** Industrial design language with mono-spaced typography and terminal-like prompts (`INITIALIZING PORTFOLIO_`).
- **Custom Cursor:** Interactive, physics-based custom cursor with hover state scaling.
- **Smooth Animations:** High-performance marquee loops, fade-in scroll reveals, and 3D hover effects.
- **Responsive Layout:** Fully optimized for mobile, tablet, and desktop environments.

## 💻 Getting Started

Want to run this locally? Follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Initialize Protocol**
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
